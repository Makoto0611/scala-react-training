## 課題

昨日定義した `Impression` case class を使う。

### Part 1：基本操作

```scala
val impressions = List(
  Impression("ad_001", 1700000000L),
  Impression("ad_001", 1700000010L),
  Impression("ad_002", 1700000020L),
  Impression("ad_003", 1700000030L)
)
```

実装する処理：
```
1. adId が "ad_001" のものだけ抽出する
2. 全インプレッションの adId だけの List を作る
3. adId が "ad_002" または "ad_003" のものを抽出する
4. 抽出した adId を大文字に変換した List を作る
```

### Part 2：メソッドチェーン

Part 1 の 3 と 4 を **1行のメソッドチェーン** で書き直す。

### Part 3：広告マスターとの組み合わせ

```scala
val ads = List(
  Ad("ad_001", "テスト広告A", 50000),
  Ad("ad_002", "テスト広告B", 30000)
)
```

`impressions` の中から `ads` に存在する `adId` のものだけ抽出する。
（`ad_003` は `ads` に存在しないので除外される）
