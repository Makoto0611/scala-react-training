### Part 2：実装

`src/main/scala/models/AdModels.scala` に3つの `case class` を定義する。

**要件：フィールド名・型は自分で決める**

定義すべきデータ：
```
広告（Ad）
  → 広告を一意に識別できる
  → 広告名を持つ
  → 予算（円）を持つ

インプレッション（Impression）
  → どの広告が表示されたか識別できる
  → いつ表示されたか（Unix時間）

クリック（Click）
  → どの広告がクリックされたか識別できる
  → いつクリックされたか（Unix時間）
```
