# 課題2 - 広告パフォーマンスダッシュボード

**目安時間**: 6〜8時間
**参照リソース**: react.dev / TypeScript公式

step1〜3で学んだ全ての知識を統合して、動く成果物を完成させる。

---

## 仕様

`src/project3-dashboard/` に実装する。

### 表示内容

```
[広告パフォーマンスダッシュボード]

フィルター: [広告ID入力欄____] [クリア]
ソート: [IMP数] [クリック数] [CTR]

| 広告ID  | IMP数  | クリック数 | CTR   |
|---------|--------|-----------|-------|
| ad_001  | 1,000  | 50        | 5.00% |
| ad_002  |   800  | 12        | 1.50% |

※ クリックで選択・再クリックで解除（行をハイライト）
```

### データソース

phase1の課題1で生成した `ads.json` を参照する。
フォーマットは以下の通り：

```json
[
  {"adId": "ad_001", "impressions": 2, "clicks": 1, "ctr": 50.00}
]
```

### 入札ボタン（phase3の繋ぎこみ用）

各行に「入札する」ボタンを追加する。
この段階ではクリックしても `console.log` するだけでよい。
phase3で Scala API との繋ぎこみを実装する。

---

## コンポーネント分割要件

最低でも以下のコンポーネントに分割すること：

```
App.tsx            → データ取得・状態管理
FilterInput.tsx    → テキストボックス・クリアボタン
SortButtons.tsx    → ソートキー選択ボタン
AdTable.tsx        → テーブル全体
AdTableRow.tsx     → テーブルの1行（入札ボタン含む）
```
