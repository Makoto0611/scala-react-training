## デモシナリオを作る

3/30の発表で使うデモの流れを決めて、`demo-script.md` に書く。

### 推奨シナリオ

```
1. 課題1を実行する（sbt run）
   → ads.json が生成される
   → ターミナルに集計結果が表示される

2. ダッシュボードを開く（localhost:5173）
   → ads.json のデータが表示される
   → フィルター・ソートが動くことを見せる

3. Scala API を起動する（sbt run）
   → localhost:8080 で起動する

4. ダッシュボードから「入札する」ボタンをクリック
   → 入札結果が画面に表示される
   → 入札なしケース（floorPrice を高くする）も見せる
```

### demo-script.md に書く内容

```
- デモの流れ（箇条書き）
- 各ステップで事前に準備しておくこと
- 説明する技術的なポイント（Scalaのここが面白い、等）
- 想定される質問と回答
```
