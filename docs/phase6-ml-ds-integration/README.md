# Phase6 - ML・DSとの連携実践

## このphaseのゴール

**データサイエンティスト（DS）や ML エンジニアが作ったモデルを、
フロントエンドエンジニアとしてプロダクトに組み込める状態になること。**

モデルを自分で作る必要はない。
「何を渡して何が返るか」「なぜその値か」が
DSとの会話の中で理解できるレベルを目指す。

---

## なぜこのphaseが必要か

adtech の DSP では以下のようなフローが実際の業務で発生する。

```
DS:「このセグメント条件で推定リーチを計算するAPIを作りました」
FE（自分）:「パラメータは何を渡せばいいですか？戻り値の型は？」
DS:「segmentId と dateRange を渡すと、reach: Int と confidence: Float が返ります」
FE（自分）:「わかりました。useQuery で実装します」
```

このやりとりが詰まらずできることがゴール。

| Phase | 内容 | Phase6 との関係 |
