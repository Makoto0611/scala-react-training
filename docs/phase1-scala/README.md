# Phase1 - Scala基礎

## このphaseのゴール

**課題1（広告ログ集計CLIツール）が完成して `sbt run` で動くこと。**

step1〜6を順番に進めながら必要な知識を身につけ、
最後に課題1として全部を統合する。

---

## 学習サイクル

```
書籍で読む（30分）
    ↓
自分で実装する（数時間）
    ↓
完了条件を確認する
    ↓
次のstepへ
```

- 実装中にAIは使わない
- 詰まったら書籍・ヒントの順で調べる
- 実装後にAIでレビューしてもらう
