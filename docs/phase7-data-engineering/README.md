# Phase7 - データエンジニアリング実践

## このphaseのゴール

**ETLパイプライン・大規模SQL・dbt の実践レベルの理解を深め、
データエンジニアとの会話で詰まらない状態になること。**

Phase5ではDatabricks/BigQueryの概念を学んだ。
Phase7ではより実践的なクエリ設計・パイプライン設計を扱う。

---

## 前提スキルとの接続

| 既存スキル | 習得タイミング |
|-----------|--------------|
| Airflow（DAG・ETL・スケジューリング・BigQuery連携） | Phase5 step1 で概念理解済み |
| BigQuery 基礎SQL・Load処理 | Phase5 step1・2 で概念理解済み |
| Databricks / Delta Lake 概念 | Phase5 step1 で学習済み |

Phase5の概念理解を土台として、本番運用レベルに引き上げる。

---

## step一覧

| step | テーマ | 目安時間 |
|------|--------|---------|
| step1 | BigQuery 実践SQL（adtech文脈での集計パターン） | 2時間 |
| step2 | ETLパイプライン本番運用パターン（Airflow深化） | 2時間 |
| step3 | dbt 基礎（SQLの変換管理とドキュメント化） | 1.5時間 |
| step4 | Tableau 連携の概念（データソース設計パターン） | 1時間 |

**合計目安: 6〜7時間**
